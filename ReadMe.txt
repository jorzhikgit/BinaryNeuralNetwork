*** Artificial Neural Network based on Binary Matrix ***

This is a project for demonstrating the binary matrix learning process.
In this version of a neuron the horizontal bar is located at the top of the neuron, though in newer version it is on the bottom.
Visual Studio 2017 C# is required.
Please notice the BigInteger and BigRational modules for big numbers of unlimited size are used.
For demo start MKS console application. Watch the Visual Studio Debug window to check for metrics change to 0.
Any questions: akazant@gmail.com

Демо проект показывающий процесс обучения бинарной матричной нейронной сети.
В данной версии нейрона горизонтальная перегородка расположена сверху от нейрона, хотя в более поздних версиях она снизу.
Проект для Visual Studio 2017 C#.
Обратите внимание на использование модулей BigInteger и BigRational для работы с большими числами неограниченной величины.
Для запуска демо назначьте запускаемым консольное приложение MKS и запустите его. За уменьшением значения метрики до нуля в процессе обучения матрицы можно следить в окне Visual Studio - Debug. 
вопросы по email: akazant@gmail.com
